fileexchange
============

This is an example of local file exchange hub (like rapidshare.com).

Software used:

- _nginx_ to handle file uploads/downloads;
- _gunicorn_ to serve wsgi app;
- _flask microframework_ to build website;
- _sqlalchemy_ as ORM;

Tested in browsers:

- Google Chrome 22.0.1229.94m;
- Opera 12.02
